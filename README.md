# BISF
a mac changer, only for linux
Features
Manual Mode: Allows you to specify a custom MAC address.
Random Mode: Automatically generates a valid, random MAC address.
Change Validation: Verifies the integrity of the process using regular expressions to ensure the change was applied correctly

Requirements
Linux-based operating system.
Sudo to interact with the link layer.

Comand help
python BISF.py --help
